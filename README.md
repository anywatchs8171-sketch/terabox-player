# Terabox Instant Player

Watch any Terabox video instantly without login.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/anywatchs8171-sketch/terabox-player)

1. Paste any Terabox link
2. Click Play
3. Video starts instantly!
